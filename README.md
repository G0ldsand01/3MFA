# 3MFA - 3D Printing, Rendering, and IT Services
Welcome to the 3MFA GitHub repository! We are a company specializing in 3D printing, visual rendering, and 3D and IT services. This repository contains resources, information, and guides related to our projects and services.

## 🛠️  Our Services
### 1. 3D Printing
 - **Rapid prototyping**
 - **Custom models**
 - **Small-batch production**
 - **Available materials: PLA, TPU, PETG, others available on request (additional fees)**
### 2. Visual Rendering
Modeling: creation of realistic 3D models for any type of project.
Post-Production: retouching and enhancing renders for optimal quality and a perfect model before printing.
 ### 3. IT Services
 - **Consulting and advice for IT and 3D projects.**
## 🚀 *Technology and Skills*
We use the latest technologies to ensure high-quality results:
 - **Modeling and rendering software**: Blender, Fusion 360, ThinkerCAD.
 - **3D printers**: BambuLab, and Creality.
 - **Available printers**: BambuLab A1 (2), BambuLab P1P(1), BambuLAB A1 mini(1), Creality Ender 3 v2(1), Creality Ender 3 V2 se.
- **Software development**: HTML, Java, JavaScript, C++, Svelte.
- **Services**: Computer configuration, CUSTOM computer assembly, Printing service for your own models, and much more.
